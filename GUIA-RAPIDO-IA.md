# 🚀 Guia Rápido - Configurar IA em 2 Minutos

## 1️⃣ Obter Chave API do Groq (Grátis)

1. Acesse: **https://console.groq.com**
2. Clique em **"Sign Up"** (criar conta gratuita)
3. Após login, vá em **"API Keys"** no menu
4. Clique em **"Create API Key"**
5. Copie a chave (começa com `gsk_`)

## 2️⃣ Configurar no Projeto

1. **Copie o arquivo de exemplo:**
   - Copie `env.example` e renomeie para `.env`

2. **Edite o arquivo `.env`:**
   - Encontre a linha: `GROQ_API_KEY=`
   - Cole sua chave: `GROQ_API_KEY=gsk_sua_chave_aqui`

3. **Salve o arquivo**

## 3️⃣ Testar

1. O servidor deve reiniciar automaticamente
2. Acesse: http://localhost:5173
3. Faça login (qualquer email/senha)
4. Clique em **"Assistente IA"** no menu
5. Faça uma pergunta!

**Exemplo de pergunta:**
- "Como aumentar minhas vendas no TikTok Shop?"
- "Dicas para criar vídeos virais"
- "Como fazer lives que convertem?"
